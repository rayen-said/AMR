# AGRINEX CONSTITUTION

Every generated file must follow:

- Next.js 15 App Router
- TypeScript strict mode
- TailwindCSS v4
- Shadcn UI
- Framer Motion
- Mobile-first design

Priorities:

1. Maintainability
2. Scalability
3. Accessibility
4. Performance
5. Design consistency

Never generate:

- inline styles
- duplicated code
- large components
- hardcoded colors

Always use the Agrinex Design System.
